How to add this row to your CSV on GitHub

  1. Click Generate Row above, then Copy to Clipboard.
  2. Go to your CSV file on GitHub ↗.

  4. Scroll to the bottom, paste your new row on a new line.
  5. Click Commit changes — done! 🎉
